== DESAFIO SERVICENET

Para rodar a aplicação é necessário instalar o ruby e o rails.

Depois de baixar o projeto, executar o seguinte comando dentro da pasta do projeto:
<tt>bundle install</tt>

Após isso:
<tt>rake db:migrate</tt>

E por fim,
<tt>rails server</tt>

O banco de dados é local com sqlite então não precisa se preocupar.

Qualquer dúvida entra em contato.
